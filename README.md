# python-bunker-rpg-console
# 🎮 Бункер: Текстовая RPG на Python

Простая, но увлекательная консольная выживалка, написанная на Python. Игрок оказывается в заброшенном бункере, из которого нужно сбежать, обыскивая комнаты в поисках электронных чипов и выживая в суровых условиях.

## 📝 Особенности проекта
* **Случайная генерация:** Каждая комната наносит случайный урон и имеет рандомный шанс дропа полезного лута или хлама.
* **Система инвентаря и предметов:** Отслеживание собранных чипов и аптечек с помощью списков Python.
* **Механика лечения:** Возможность восстановить здоровье найденными аптечками, но не выше максимального лимита в 100 HP.
* **Защита от некорректного ввода:** Игра не ломается и не тратит ход игрока, если введена опечатка.
* **Проверка условий:** Динамическое отслеживание здоровья персонажа в реальном времени и фиксация финала при его обнулении.
* **Атмосферные задержки:** Паузы во время поиска создают эффект реального исследования комнат бункера.

## 🕹️ Как играть
1. Запустите скрипт.
2. Выбирайте действия:
   * `искать` — обыскать комнату (тратит HP, есть шанс найти чип или аптечку).
   * `лечиться` — использовать найденную аптечку, чтобы восстановить +30 HP.
   * `ворота` — попытаться открыть выход (нужно накопить 3 чипа).
   * `сдаться` — досрочно завершить игру.
3. Попробуйте выбраться живым!
