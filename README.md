# python-bunker-rpg-console
# 🎮 Бункер: Текстовая RPG на Python

Простая, но увлекательная консольная выживалка, написанная на Python. Игрок оказывается в заброшенном бункере, из которого нужно сбежать, обыскивая комнаты в поисках электронных чипов.

## 📝 Особенности проекта
* **Случайная генерация:** Каждая комната наносит случайный урон и имеет рандомный шанс дропа чипа.
* **Система инвентаря:** Отслеживание собранных предметов с помощью списков Python.
* **Защита от некорректного ввода:** Игра не ломается и не тратит ход игрока, если введена опечатка.
* **Проверка условий:** Динамическое отслеживание здоровья персонажа в реальном времени.
* **Атмосферные задержки:** Паузы во время поиска создают эффект реального исследования комнат бункера.

## 🕹️ Как играть
1. Запустите скрипт.
2. Выбирайте действия:
   * `искать` — обыскать комнату (тратит HP, есть шанс найти чип).
   * `ворота` — попытаться открыть выход (нужно накопить 3 чипа).
   * `сдаться` — досрочно завершить игру.
3. Попробуйте выбраться живым!
